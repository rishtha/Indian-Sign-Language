# Indian Sign Language Recognition using Deep Learning

A deep learning model for recognizing and interpreting Indian Sign Language (ISL) hand gestures, published at the IEEE International Conference on Communication and Signal Processing (I2CT) 2022.

## Publication
**Hand Gesture Interpretation for Indian Sign Language Using Deep Learning**  
IEEE I2CT 2022 — DOI: [10.1109/I2CT54291.2022.9825322](https://doi.org/10.1109/I2CT54291.2022.9825322)

## Overview
This project builds a gesture recognition pipeline that classifies ISL hand gestures using a convolutional neural network (CNN) trained on a labeled gesture dataset. The model processes hand gesture images and maps them to corresponding ISL signs.

## Tech Stack
- **Language:** Python
- **Frameworks:** TensorFlow / Keras
- **Libraries:** OpenCV, NumPy, Matplotlib
- **Tools:** Jupyter Notebook

## Features
- Real-time hand gesture detection and classification
- CNN-based architecture optimized for gesture recognition
- Trained and evaluated on a custom ISL gesture dataset
- Achieves high classification accuracy across ISL alphabet signs

## Results
Model performance and evaluation metrics are documented in the published IEEE paper linked above.
