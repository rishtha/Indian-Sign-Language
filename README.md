Indian Sign Language Recognition using Deep Learning
A deep learning model for interpreting Indian Sign Language (ISL) gestures, published at IEEE I2CT 2022.
Tech stack: Python, TensorFlow/Keras, OpenCV
What it does:
Classifies hand gestures corresponding to ISL alphabets and words using a CNN-based architecture trained on a custom dataset.
Publication:
Hand Gesture Interpretation for Indian Sign Language Using Deep Learning — IEEE I2CT 2022
DOI: 10.1109/I2CT54291.2022.9825322
